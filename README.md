# 📦 LogiChain: Industrial Blockchain Logistics Tracker

LogiChain is a professional, blockchain-powered logistics and supply chain tracking system. It ensures transparency, data integrity, and authenticity for products moving from manufacturers to consumers.

## 🚀 Features

- **Product Registration**: Manufacturers can register new products into the blockchain with a unique immutable ID.
- **Real-time Tracking**: Track the movement and location transitions of any asset.
- **Ownership Transfer**: Securely transfer responsibility from Manufacturer → Distributor → Seller.
- **Authenticity Verification**: Verify if a product is an original part of the blockchain records.
- **Chain Audit**: Validate the entire blockchain integrity to ensure no data has been tampered with.
- **Immutable Ledger**: Uses SHA-256 hashing and a Proof-of-Work mechanism to secure data.

## 🛠 Technology Stack

- **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3.
- **Icons**: Lucide Icons.
- **Typography**: Outfit & JetBrains Mono (Google Fonts).
- **Blockchain**: Custom implementation using Web Crypto API.

## 📊 How it Works

1. **Blocks**: Each logistics event (registration, transit, etc.) is stored in a block.
2. **Hashes**: Each block contains its own hash and the hash of the previous block, creating a secure chain.
3. **Mining**: A mining process (Difficulty 4) simulates computational effort to add a block.
4. **Validation**: The system checks if hashes match across the entire chain to detect any manipulation.
